## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **SEO**: React Helmet Async + Structured Data
- **Backend**: Node.js + Express
- **Database**: Supabase
- **Authentication**: Google OAuth + Supabase Auth

## 📦 Installation

```bash

# Start development server
npm run dev

# Build for production
```
